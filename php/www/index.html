常用操作

Docker 命令

# 查看容器状态
docker-compose ps

# 查看日志
docker-compose logs -f dm8
docker-compose logs -f php-dm

# 进入容器
docker-compose exec dm8 bash
docker-compose exec php-dm bash

# 重启服务
docker-compose restart

# 停止服务
docker-compose down

PHP连接示例

<?php
// PDO方式
$pdo = new PDO('dm:host=dm8;port=5236;dbname=SYSTEM', 'SYSDBA', '123456');

// 或者使用dm扩展
$conn = dm_connect('dm8:5236', 'SYSDBA', '123456');
?>

注意事项

  • 请确保已放置达梦数据库授权文件
  • PHP达梦扩展需要从达梦官网下载
